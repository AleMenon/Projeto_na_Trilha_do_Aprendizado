<table>

Esse comando permite organizar dados em forma de tabela, e para isso usa-se a tag <table> para abrir a tabela, e então, inserimos as tags <tr> para iniciar uma linha, <th>, que é usada para fazer o cabeçalho destacado, e por fim <td>, usado para inserir o resto das informações da tabela.

<del> e <ins>

As duas tags são bem similares, resumindo, o <del> serve para cortar uma palavra ao meio e o <ins> para dar destaque, sublinhando o texto.

Sintaxe

Gosto de java HTML.

<sub>

Essa tag tem a função de deixar o texto menor e um pouco deslocado na linha.

Sintaxe

Qual palavra está usando a tag <sub>?

<blockquote> e cite

Primeiramente, sobre o <blockquote>, é usado para fazer uma citação longa, e na própria tag é usado o cite para referenciar o site de onde foi tirado a citação.
Exemplo: <blockquote cite="link usado"> Citação </blockquote>

Sintaxe

Citação tirada de w3schools:

O Elemento HTML <blockquote> (ou Elemento HTML de citação de bloco) indica que o texto incluído é uma longa citação.

<q>

<abbr>

Essa tag é usada para marcar abreviações, deixando elas com um sublinhado pontilhado, e dentro da própria tag usa-se o title para descrever a abreviação usada.
Exemplo: <abbr title="Doutor"> Dr. </abbr>

Sintaxe

Dr. Juliano

<address>

Usado para marcar o endereço de contato de um autor de um determinado arquivo ou documento. O texto marcado com essa tag ficará necessariamente em itálico.

Sintaxe

Alexandre Alberto Menon
Email

<bdo>

Usado para inverter o texto escrito usando dentro da própria tag dir=ltr de "left to right" e dir=rtl de "right to left".

Sintaxe

Direcionamento "left to right".
Direcionamento "right to left".

<code>

Essa tag é usada para definir uma parte de um código de computação, mudando para uma fonte expecífica.

Sintaxe

Usando a tag <code> para marcar um texto.

<pre>

Essa tag é usada para marcar um texto e deixa-lo do jeito que foi enviado, sem precisar formatar ou algo do tipo

Sintaxe


            Sim, tenho saudades.
            Sim, acuso-te porque fizeste
            o não previsto nas leis da amizade e da natureza
            nem nos deixaste sequer o direito de indagar
            porque o fizeste, porque te foste

O texto acima foi escrito sem o uso de qualquer tag além do <pre>.

<kbd>

Essa tag é parecida com o <code> citado acima, mas é usada para marcar alguma função ou tecla do teclado.

Sintaxe

Ctrl + C copia alguma informação.

Ctrl + V cola a informação copiada.
